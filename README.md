# Finance with Kellie — Product Demo

Interactive walkthrough of the Kellie Finance Bot user journey (6 stages, narrated).

## Quick deploy to GitHub Pages

1. Create a new repo on GitHub (e.g. `kellie-bot-demo`).
2. Upload **`index.html`** to the repo root.
3. Go to **Settings → Pages → Source: Deploy from branch → `main` / root**.
4. Your demo goes live at `https://<your-username>.github.io/<repo-name>/` within ~1 minute.

## Notes for viewers

- **Click ▶ to start narration.** Browsers block auto-play of audio until you interact with the page.
- **Voice quality varies by browser.** Chrome/Edge use Google US English (highest quality). Safari uses Samantha (macOS). Firefox uses the system default.
- **Works offline** — the entire demo is self-contained in one HTML file.
- **Best on desktop.** The phone mockup is the focal point; rails on either side provide context.
